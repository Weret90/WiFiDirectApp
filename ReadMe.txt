Приложение по отправке сообщений через WiFi-Direct

1. Могут быть связаны только два устройства
2. Вай-фай должен быть включен. Если он будет выключен появится сообщение с предложением его включить. Также рекомендуется отключить мобильную точку доступа
3. Для старта нужно нажать на ОБОИХ телефонах кнопку "Искать устройства"
4. В случае успешного поиска на экране появится название устройства. При клике по нему должно произойти соединение устройств
5. Если на экране сверху появится надпись связь установлена - можно отправлять сообщения
6. Отправленные и принятые сообщения будут выводиться на экран
7. При сворачивании или закрытии приложения связь разрывается, нужно будет повторить процедуру установки связи
8. Сообщения будут сохраняться в базу данных телефона
